DATA SOURCES AT A GLANCE
================================================================================

ESSENTIAL (MVP):
  • Synthea (Synthetic EMR): https://synthetichealth.github.io/synthea/
    └─ Generates: EMR patient, encounter, diagnosis, procedure data
    └─ Time: 15 min setup, generates 1K patients in <5 min
    └─ Cost: Free

  • CMS NPPES (Real providers): https://download.cms.gov/nppes/NPI_Files.html
    └─ Provides: 10-digit NPI provider directory
    └─ Time: 5 min download, ~1.5 GB compressed
    └─ Cost: Free

  • Synthetic Claims (You generate): Custom Python script
    └─ Maps: Synthea encounters to RCM claims
    └─ Time: 1-2 hours to build script
    └─ Cost: Free

  • Synthetic Referrals (You generate): Custom Python script
    └─ Generates: Referral orders with counterparties
    └─ Time: 1-2 hours to build script
    └─ Cost: Free

OPTIONAL (Validation):
  • MIMIC-IV (Real clinical): https://physionet.org/content/mimiciv/3.1/
    └─ Provides: De-identified ICU/ED data for validation
    └─ Time: 2-4 hours (training + DUA)
    └─ Cost: Free (requires Data Use Agreement)

REFERENCE:
  • X12 835 (Claims standard): https://www.cms.gov/medicare/billing/electronicbillingeditrans/downloads/835-flatfile.pdf
    └─ Specifies: Remittance advice format
    └─ Time: 1 hour to review
    └─ Cost: Free
